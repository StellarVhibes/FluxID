# FluxID Frontend 🎨

This directory contains the **Next.js** PWA frontend for FluxID — the Liquidity Identity Layer on Stellar.

## 📚 Documentation

- **[Frontend Issues Tracker](../docs/ISSUES-FRONTEND.md)**: Pick a task here.
- **[Integration Guide](../docs/FRONTEND_GUIDE.md)**: Learn how to connect Freighter.

## 🚀 Quick Start

```bash
npm install
npm run dev
```

## Features

- Wallet connection via Freighter
- Real-time liquidity score display
- Flow visualization dashboard
- Risk indicator (Low/Medium/High)
- Actionable financial suggestions

---

*Built with Next.js, TypeScript, and Tailwind CSS.*
